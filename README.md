## 📖 **About The Project**

**SmartResume AI** is a modern, AI-powered resume builder application designed to help job seekers create professional, ATS-friendly resumes that stand out to recruiters.

Unlike traditional resume builders, SmartResume AI leverages **Google Gemini AI** to provide intelligent suggestions, optimize content for Applicant Tracking Systems (ATS), and help users tailor their resumes to specific job descriptions.

### **Why SmartResume AI?**

- 💡 **AI-Powered Suggestions** - Get real-time feedback to improve your resume
- 📊 **ATS Optimization** - Increase your chances of getting past automated screening
- 🎨 **Beautiful Templates** - Professional designs that impress recruiters
- 🔒 **Privacy First** - Your data stays secure
- 🚀 **Fast & Responsive** - Works seamlessly on all devices

## 🛠️ **Built With**

### **Frontend**

- [Next.js 14](https://nextjs.org/) - React framework with App Router
- [TypeScript](https://www.typescriptlang.org/) - Type safety
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first styling
- [Axios](https://axios-http.com/) - HTTP client
- [React Hook Form](https://react-hook-form.com/) - Form management
- [html2canvas](https://html2canvas.hertzen.com/) - PDF generation
- [jsPDF](https://github.com/parallax/jsPDF) - PDF export

### **Backend**

- [Node.js](https://nodejs.org/) - JavaScript runtime
- [Express](https://expressjs.com/) - Web framework
- [MongoDB](https://www.mongodb.com/) - Database
- [Mongoose](https://mongoosejs.com/) - ODM
- [Multer](https://github.com/expressjs/multer) - File upload
- [Google Generative AI](https://ai.google.dev/) - Gemini AI

### **AI Integration**

- [Google Gemini 1.5 Flash](https://ai.google.dev/) - Fast, efficient AI model
- 60 requests/minute free tier
- Context-aware resume analysis
- Job description matching

### **Development Tools**

- [Nodemon](https://nodemon.io/) - Auto-restart during development
- [Git](https://git-scm.com/) - Version control
- [npm](https://www.npmjs.com/) - Package management
